## 프로젝트 개요
이 문서는 자동화된 PR 관리 시스템의 핵심 기능을 설명합니다.

### 주요 기능
- GitHub PR 자동 생성 및 관리
- AI 기반 코드 리뷰 지원
- 다국어 지원 (한국어/영어)
- 충돌 해결 가이드 제공

## 기술 스택
### 백엔드
- Node.js
- TypeScript
- OpenAI API 통합
- GitHub API 통합

### 개발 도구

- Prettier
- Vitest
- Zod

2. 의존성 설치

## 사용 예시
```bash
# PR 생성
autopr new

# 코드 리뷰
```
